# Handleiding installatie DC1
Prerequisites:
- Virtualbox software
- Voldoende RAM geheugen om VM's te runnen


## Stap 1: VM aanmaken & configureren
**BELANGRIJK**: Deze VM vereist dat `WIN-DC1` en `WIN-SQL-SCCM` up en running is om correct ingesteld te worden!

Belangrijke `instellingen` voor deze VM:
- Deze client wordt geboot via `WIN-SQL-SCCM`, het is dus belangrijk dat onder het tabblad `Systeem`, onder "opstartvolgorde" `Netwerk` bovenaan en aangevinkt staat.

Vervolgens start je de VM op en doorloop je de Windows 10 installatie.
