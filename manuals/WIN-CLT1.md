# Handleiding installatie DC1
Prerequisites:
- Virtualbox software
- Windows 10 ISO file
- Voldoende RAM geheugen om VM's te runnen


## Stap 1: VM aanmaken & configureren
**Opmerking**: Deze VM vereist dat `WIN-DC1` up en running is om correct ingesteld te worden.

Belangrijke `instellingen` voor deze VM:
- 1 Netwerkadapter: Host-Only Adapter; deze zal via DHCP automatisch ingesteld worden
- De VM installeren adhv. een ISO file (zoals bij `WIN-DC1`...)
    - Nu is de ISO de Windows 10 versie
